3. 配置

在大多数系统中，默认私钥（~/.ssh/id_rsa 和 ~/.ssh/identity）会自动添加到 SSH 身份验证代理中。应无需运行 ssh-add path/to/key，除非在生成密钥时覆盖文件名。
