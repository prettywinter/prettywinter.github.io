一、常用命令

npm install 可简写为 npm i

二、Yarn

npm 是 Node 自带的包管理工具，使用起来很方便。也可以使用 yarn，yarn 是 facebook 发布的一款取代npm的包管理工具，详情请戳 yarn官网介绍

三、pnpm

pnpm 是微软发布的一款 Node 平台的包管理工具，用的人数也挺多的。默认使用 npm 的安装源。

官方文档https://pnpm.io/
